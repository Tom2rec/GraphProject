# Graph Theory - project


## Implementation of Bellman-Ford's algorithm


## Graph format 

```json
{
  "start":  number,
  "end": number,
  "weight": number
}

// start   <- of the edge 
// end     <- of the edge
// number  <- random number
// (int)   <- start, end
// (float) <- weight
```

## Usage

```bash
C:\GraphProject> python run.py
```
